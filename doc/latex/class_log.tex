\doxysubsection{Detailed Description}
Logs the occurred events with additional information and status. 

This static class is made for logging events where actions might contain important messages. Used for tracking control flow, errors, etc. If you need more information about the log levels and when to use them, check out \href{https://stackoverflow.com/questions/2031163/when-to-use-the-different-log-levels}{\texttt{ this}} link.
