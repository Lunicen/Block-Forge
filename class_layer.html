◆ OnEvent()

virtual void Layer::OnEvent ( HumanInterfaceDevice &  hid)
pure virtual

This method is used to handle events.

This method should be used when you want to declare functionality that is strongly related to this layer. For example, when you are playing a simulation and you pause the game. Then you may want to freeze input events that are related to this layer, but you would like to render the content of this layer.

Using this method, you can tell it to freeze when necessary and use the OnUpdate() method to render continuously.

Parameters
hid- Reference to the Human Interface Device.

Implemented in PauseMenuLayer, and SandboxLayer.
